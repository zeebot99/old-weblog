# old-weblog

> [!WARNING]
> This website is depreciated.

This is a repository to store my old omg.lol weblog files. Here's what it used to look like:

![The homepage of my old weblog](https://github.com/zeebot99/old-weblog/blob/main/old-weblog.png)

This repo is ***not in active development***. I have moved to [11ty](https://11ty.dev) for static site generation and GitHub Pages for hosting. You can see the fruits of my labor at [zeebot.weblog.lol](https://zeebot.weblog.lol) and the repo for that [here](https://github.com/zeebot99/web-blog).

## Contributing

This repo is for my documentation purposes only. If you want a similar website, you can sign up to get your own domain at [omg.lol](https://home.omg.lol) (this is not an affiliate link, but you can let them know that [zeebot sent you](https://home.omg.lol/referred-by/zeebot)) or make your own based on the files in this site. If you plan to use the second method, I would recommend [forking this repo](https://github.com/login?return_to=%2Fzeebot99%2Fold-weblog).

## Support

You can reach me at any of the methods listed on my [contact page](https://zeebot.weblog.lol/contact).
